# nft-hero

Movie discovery UI with themed movie details and ticket booking flow.

## Features

- Landing page with clickable movie cards
- Movie detail pages with synopsis, cast, gallery, and reviews
- Ticket booking UI with date/time and seat selection
- Booking summary with total pricing

## Run

```bash
npm install
npm run dev
```

## Build and lint

```bash
npm run build
npm run lint
```

## Key files

- `src/components/MovieDetailPage.jsx`
- `src/components/BookingPage.jsx`
- `src/components/CardCarousel.jsx`
- `src/data/movies.js`
