# Add your images here

Place your images in this folder to use them on the landing page:

- **hero.jpg** - Main banner (1200x500 recommended)
- **movie1.jpg, movie2.jpg...** - Movie posters (300x450 recommended)
- **event1.jpg, event2.jpg...** - Event images (400x250 recommended)
- **premier1.jpg, premier2.jpg, premier3.jpg** - Premiere posters (300x450 recommended)

Until you add images, Unsplash placeholders will be used automatically.
